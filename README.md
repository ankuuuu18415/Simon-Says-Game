# Simon Says Game 🎮

A web-based Simon Says memory game built with HTML, CSS, and JavaScript. Challenge your memory with increasing levels of difficulty!

## Features
- Interactive colored buttons
- Level-based difficulty
- Visual feedback and animations
- Score display on game over

## How to Play
1. Open `index.html` in your browser
2. Press any key to start the game
3. Repeat the color sequence by clicking the buttons
4. Try to reach the highest level possible!

## Technologies Used
- HTML
- CSS
- JavaScript
